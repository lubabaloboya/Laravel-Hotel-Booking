# Laravel 5.5 based system for hotel managers to manage room bookings

## How to use

- Clone the repository
- MySQL create database name: laravel_hotel_booking
- Update you MySQL database username and password are correct 
- Open CMD
- Run composer install
- Run php artisan key:generate
- Run php artisan migrate --seed (it has some seeded data for your testing)
- Navigate to URL 
- username / Emails : admin@admin.com 
- password : password
